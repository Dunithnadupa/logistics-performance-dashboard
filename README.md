📦 Logistics Performance & Cost Efficiency Dashboard

📊 Project Overview

• This project focuses on analyzing logistics delivery operations using interactive dashboards to assess performance, cost efficiency, and environmental impact. The analysis was conducted as a group project for the university module IS 3005 – Statistics in Practice.

🎯 Objectives

✅ Evaluate delivery success rates and delay patterns

🚚 Analyze cost efficiency across regions, vehicle types, and delivery modes

🌱 Measure sustainability and environmental impact

📈 Provide data-driven insights for logistics optimization

⭐ Key Features

•  Advanced DAX measures for performance tracking
• Interactive visualizations for regional and partner analysis
• Cost and delay analysis by vehicle type and weather conditions
• Sustainability scoring and efficiency metrics
• Custom Measures (DAX)
• Cost Percentage
• Delay Percentage
• Delay Rate
• Delivery Success Rate
•  Dynamic Gauge Maximum
